R and Python are programming languages. Each offers powerful capabilities for research and data analysis. Both languages are open source. Each has a wide community of users and resources for learning. Some people learn both languages, while others rely on one. While Python offers general utility for research with data, R is used for more focused statistical analysis. Many researchers underestimate how significant programming skills may be to enable them to reach their goals.

Common Researcher Pitfalls: Python or R

Researchers often make the following miscalculations:

  • Underestimate how much time it will take to learn Python or R.
  • Underestimate how soon their research will require these tools.
  • Assume that their level of skill with Python or R is adequate to achieve their research goals.

How to Learn Python or R

There are a variety of ways to learn these tools. No one way is the "right" way. Choose an approach that best fits how you learn and the time and resources you have available.

Ways to Learn

Take a class at CMU:
  • Computer Science
  • Heinz
  • Tepper
Attend a workshop with CMU University Libraries:
  • University Libraries host free workshops for researchers using the Software Carpentries curriculum.
Use books and other materials for self-study:
  • The book R for Data Science is formally available in a free online version.
  • The book Learn Python the Hard Way is available as an electronic text through CMU libraries.
  • Follow the Software Carpentries curriculum independently, and use this skills-assessment module to test your knowledge.
Take an online courses:
  • LinkedIn Learning: Courses are free to CMU students, faculty and staff, and include multiple options for learning R or Python.
  • CMU’s Open Learning Initiative (OLI) provides a free Python course: Principles of Computation with Python — Open & Free
  • Coursera offers a variety of options for learning R or Python, and many courses can be audited free.
  • edX offers a variety of options for learning R or Python, and courses can be audited free.
Other resources:
  • Google offers a Python class, a free class “for people with a little bit of programming experience who want to learn Python.”
  • swirl: swirl is a software package for the R programming language that turns the R console into an interactive learning environment.
  • Mark Gardener is an author and consultant who provides a guide to “Getting Started with R” on his website.
  • Paul van der Laken is a data analyst who provides a 6-step guide for learning R on his website.
